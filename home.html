ReactJS

React is a JavaScript library that is used for building good-looking and functional user interfaces. It was developed by Facebook and is still maintained by Facebook developers. React uses code snippets called components to build a larger application. The components can be anything from a text-box to a modal window to a navigation bar. In React these components are all built in different places and then used in the final app. The benefit is that the components are reusable. If a developer is working on a list which will display in 100 boxes on a web page, they only need to write one component for the “Box” and import it to their project, pass the relevant properties (props) in and React will do the rest.

React also uses a feature call state. State is a JavaScript object which contains working information that might be updated regularly. One of the coolest things about React is that it keeps an eye on the DOM and also keeps a virtual DOM, when it senses changes to the DOM, it re-renders the page, but only the objects that have changed. It’s quick and efficient.

In React we use a modified HTML-like code called JSX. All of our components are written in JavaScript, but they are implemented in JSX which looks similar to HTML with some minor changes.
