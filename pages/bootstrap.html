Bootstrap Framework

This page uses Bootstrap CSS and jQuery script to resize automatically and fit devices based on screen size. Collectively these tools can be referred to as the Bootstrap Framework. Whether on a computer, a tablet or a smart phone, the page will optimize for the display and orientation.

Typeography & Buttons

Bootstrap has a preloaded set of typeography and buttons. All the developer needs to do is use it.

The developer can make use of special pre-defined CSS classes and HTML tags to use these tools.

Forms, Dropdowns & Navbars

Bootstrap has a library of forms and dropdown menus built in for easy integration.

Bootstrap also has many preloaded navbar designs and options that can be called.

Images & Glyphs

Bootstrap has several image options to easily create effects with minimal code.

The framework also has several graphics, called glyphs, that can be easily integrated.
